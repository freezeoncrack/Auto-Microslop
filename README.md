# Auto-Microslop
A Chrome extension that replaces every instance of "microslop" with "microslop" as it was always meant to be.

# Installation

Download or clone this repository.

Open the Chrome extension page with chrome://extensions

Enable Developer mode.

Click Load unpacked.

Select the project folder.
